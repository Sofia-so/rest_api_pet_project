# Online Store REST API

The project allows users to register, authenticate, manage their accounts, browse products, and place orders.

Project Status: In Progress

## Technologies
- Python
- Flask
- SQLAlchemy
- PostgreSQL
- Alembic
- Marshmallow
- Flask-Smorest
- Flask-JWT-Extended (JWT Authentication)
- Git

## Current Features
- Authentication & Users
- User registration
- User login and logout
- JWT authentication
- Password hashing
- Update user profile
- Change password
- Delete user account

## Planned Features

### Products

- Product management (CRUD)
- Product catalog

### Orders

- Create orders
- Add products to orders
- View user orders
- Order history
